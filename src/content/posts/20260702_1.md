# ITストラテジスト試験　修正版ロードマップ（試験日程確定版）

## ⚠️ 最初に：試験日程が正式発表されました（2026年7月6日 IPA掲載）

**当初の想定（前期試験＝2026年11月頃）より、科目A群の実施が10月中旬に前倒しされていることが判明しました。** これにより科目B群対策（特に論文）に使える時間が大きく圧縮されるため、旧プランから学習フェーズの組み方を変更します。

| 項目 | 確定日程 |
|---|---|
| 申込受付期間 | **2026年10月6日（火）〜10月24日（土）** |
| 科目A群（A-1・A-2）実施期間 | **2026年10月17日（土）〜10月27日（火）** |
| 科目B群（B-1・B-2）実施期間 | **2026年11月11日（水）〜11月23日（月・祝）** |
| 成績照会（科目B） | 2027年1月頃 |
| 合格発表 | 2027年2月頃 |
| 受験手数料 | 7,500円（変更なし） |

- 申込みの際は**科目A群と科目B群を同時に予約**します（別々には申込めません）。
- 会場・実施日ごとに空き枠が異なり、早く申し込むほど希望日を確保しやすくなります。**申込開始日（10/6）にすぐ申し込むことを強く推奨**します。
- 科目A-1試験は50分・科目A-2試験は40分（休憩10分）、科目B-1試験は90分・科目B-2試験は120分（休憩10分）で、時間・形式・出題範囲そのものに変更はありません。

### 受験日の選び方（重要な戦略判断）

科目A群・B群それぞれの実施期間の中で、自分で希望日を選んで申込みます。以下の理由から、**科目A群はできるだけ期間の早め（10/17〜10/20頃）、科目B群はできるだけ期間の遅め（11/20〜11/23頃）を狙う**ことを推奨します。

- 科目A群を早めに受ければ受けるほど、科目A試験終了から科目B群開始（11/11）までの「科目B対策に専念できる期間」が伸びます。
- 科目B群は、論文（B-2）の完成度が合否を分けるため、実施期間内で選べる最も遅い日程まで粘って対策時間を確保するのが得策です。
- 目安として、**科目Aを10/17〜10/20で受験できれば、そこから科目B受験（11/20頃）まで約4.5週間**を科目B対策に専念できます。これが今回のロードマップの前提です。

## 全体構成の変更点（旧プラン vs 新プラン）

**旧プラン**：科目A対策 → 科目B-1対策 → 科目B-2（論文）対策、と完全に順番にこなす想定（論文着手は10/27〜）。

**新プラン**：科目A対策と並行して、**論文のネタ作り・型の学習・試し書きを7月末から前倒しで開始**します。科目A試験終了後（10月下旬）から科目B試験（11月下旬）までは実質4〜5週間しかないため、この期間だけで論文をゼロから仕上げるのは危険です。早い段階で最低1〜2本は書いた経験を作っておくことが、今回の日程での合格条件になります。

```
Phase 1: 基礎固め（7/9〜7/26）
        ↓
Phase 2: 科目A対策 ＋ 論文ネタ準備・下読み（7/27〜9/6）★並行スタート
        ↓
Phase 3: 科目A総仕上げ ＋ 科目B-1入門 ＋ 論文試し書き（9/7〜10/4）
        ↓
Phase 4: 科目A直前追い込み（10/5〜10/16）
        ↓
【科目A群受験】10/17〜10/20頃を目標
        ↓
Phase 5: 科目B総仕上げ（B-1演習＋論文量産・添削）（10/21〜11/19頃）★最重要
        ↓
【科目B群受験】11/20〜11/23頃を目標
```

## Phase別ロードマップ

### Phase 1：基礎固め（7/9〜7/26、約2.5週間）

**目的**：経営戦略・IT戦略の全体像を掴む。

| 期間 | やること |
|---|---|
| 7/9〜7/12 | 試験制度・確定日程の再確認、教材の購入手配 |
| 7/13〜7/19 | 「2026年度版 ALL IN ONE パーフェクトマスター ITストラテジスト」を通読開始、経営戦略の基礎用語（SWOT・PPM・BSC等）を学習 |
| 7/20〜7/26 | DX・IT戦略系の基礎知識を学習。科目A-2過去問を1年分解いて現状把握 |

---

### Phase 2：科目A対策 ＋ 論文ネタ準備・下読み（7/27〜9/6、6週間）★並行開始が最大の変更点

**目的**：科目A-1・A-2の土台を固めつつ、論文の「ネタ」を仕込み始める。ここで論文に一切手をつけないと、11月に間に合わない。

| 期間 | やること |
|---|---|
| 7/27〜8/9 | 「専門知識＋午後問題」の重点対策の演習問題181問を1周解く／並行して「合格論文の書き方・事例集」を2〜3本読み始める |
| 8/10〜8/23 | 間違えた問題のみ複数回復習、苦手分野特定／論文をさらに2〜3本読み、自分の実務経験を棚卸しして論文ネタ候補を書き出す（メモ程度でOK） |
| 8/24〜9/6 | 科目A-1過去問を数年分サラッと確認（免除者はスキップ）／論文ネタを2〜3パターンに絞り込む「ネタ帳」の骨格を作る |

**ポイント**：論文をこの段階で完璧に書ける必要はありません。「どの実務経験をどのテーマに使えそうか」の見当をつけておくだけで、10月以降の執筆速度が大きく変わります。

---

### Phase 3：科目A総仕上げ ＋ 科目B-1入門 ＋ 論文試し書き（9/7〜10/4、4週間）

**目的**：科目Aを合格ラインまで固め、科目B-1の解答テクニックを学び、論文を実際に1〜2本書いてみる。

| 期間 | やること |
|---|---|
| 9/7〜9/13 | 科目A群の模擬試験を時間を計って実施 |
| 9/14〜9/20 | 「情報処理教科書」で科目B-1の解答テクニック（マーキング・下書き）と科目B-2の基本構成（設問ア/イ/ウ）を学ぶ |
| 9/21〜9/27 | 科目B-1の過去問を数問解いてみる／論文を実際に1本、時間を気にせず書いてみる（ネタ帳の中身を初めて文章化） |
| 9/28〜10/4 | 書いた論文を模範解答と比較して型のズレを確認／科目B-1をさらに数問解く |

**ポイント**：ここで最低1本、論文を通しで書いた経験を作ることが目標です。「書けるかどうか」を10月に初めて試すのはリスクが高すぎます。

### Phase 4：科目A直前追い込み（10/5〜10/16、2週間）

| 期間 | やること |

### Phase 5：科目B総仕上げ（10/21〜11/19頃、約4.5週間）★最重要フェーズ

**目的**：科目B-1を安定して時間内に解き切れるようにしつつ、論文を量産・添削して仕上げる。旧プランでは6週間かけていた論文対策を、ここに凝縮します。

| 期間 | やること |
|---|---|
| 10/21〜10/27 | 科目B-1の過去問を1日1問ペースで演習（残っている分をこなす） |
| 10/28〜11/3 | 論文を週2本ペースで執筆。書いたら模範解答と比較 |
| 11/4〜11/10 | 論文をさらに2本執筆。可能なら第三者添削（通信講座の単発添削サービス等）に出す |
| 11/11〜11/17 | 添削結果を反映して弱点を修正。120分計って本番形式で論文を書く練習を繰り返す |
| 11/18〜11/19 | 科目B-1・B-2を通しで本番同様にシミュレーション。最終確認 |

**チェックリスト（科目B受験前に最低限できているべきこと）★ここが唯一妥協できないポイント**
- [ ] 論文ネタが最低2パターン完成している
- [ ] そのネタで論文を最低5本以上書いた経験がある
- [ ] 120分で2,000〜3,000字を書き切れる
- [ ] 第三者に最低1回は論文を見てもらっている
- [ ] 科目B-1を制限時間（90分）内に解き切れる

**できていない場合**：11/18〜11/19の総仕上げ期間を使って、模試ではなく論文の追い込みに全振りしてください。科目Aがどれだけ良くても、論文がB評価（不合格ランク）だと総合不合格になります。

## 学習ペース早見表（19週間・修正版）

旧プランは26週間で260〜310時間（週10〜12時間）を目安にしていましたが、**準備期間が19週間に圧縮されたため、同じ総学習時間を確保するには週あたりの学習時間を増やす必要があります。**

| 1日の学習時間 | 週あたり | 19週間の総時間 | 評価 |
|---|---|---|---|
| 1時間 | 7時間 | 約133時間 | 大幅に不足 |
| 1.5時間 | 10.5時間 | 約200時間 | ギリギリのライン |
| 2時間 | 14時間 | 約266時間 | 安全圏 |
| 2.5時間 | 17.5時間 | 約333時間 | 余裕を持って合格圏 |

**平日1.5〜2時間＋休日にまとめて3〜4時間（週15時間前後）**を目安にすると、旧プランの水準（260〜310時間）に近づけられます。**旧プランの「平日1〜1.5時間」ペースでは、今回確定した日程には間に合わない可能性が高い**ので、可能な範囲で日々の学習時間を増やすことを推奨します。

## 心構え（修正版）

1. **論文対策を「10月以降」に先延ばししない**　これが今回の日程で唯一かつ最大の変更点です。7月末から論文のネタ作りに着手してください。
2. **申込みは10/6の受付開始日にすぐ行う**　科目A群はできるだけ早め、科目B群はできるだけ遅めの日程を狙うのが基本戦略です。
3. **科目Aに時間をかけすぎない**　実務経験があれば比較的取りやすいので、10月中旬の受験に向けて9月中にある程度固め、10月以降は科目Bに全振りする意識を持ってください。
4. **今後も日程・詳細（試験会場一覧、CBT体験ページなど）が追加発表される可能性がある**ため、申込み前に必ずIPA公式サイト（https://www.ipa.go.jp/shiken/2026/ap_koudo_sc_kikan.html）で最新情報を確認してください。
